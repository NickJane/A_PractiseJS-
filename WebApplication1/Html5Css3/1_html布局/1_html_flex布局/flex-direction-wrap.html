设置或检索伸缩盒对象的子元素超出父容器时是否换行
nowrap：当子元素溢出父容器时不换行。 wrap：当子元素溢出父容器时自动换行。 wrap-reverse：反转 wrap 排列。

nowrap

  • a
  • b
  • c

flex-wrap:wrap

  • a
  • b
  • c

flex-wrap:wrap-reverse

  • a
  • b
  • c
